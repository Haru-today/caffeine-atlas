CAFFEINE ATLAS 공유용 파일 안내

1. 압축 파일을 받은 사람은 먼저 zip 파일을 압축 해제합니다.
2. 압축 해제된 폴더 안에서 caffeine-sensitivity-app/index.html 파일을 더블클릭하면 홈페이지가 열립니다.
3. caffeine-sensitivity-app 폴더 안의 파일을 하나씩 따로 옮기면 이미지, 스타일, 자료실 링크가 깨질 수 있습니다.
4. 다른 사람에게 파일이 아니라 웹주소로 공유하려면 Netlify, Vercel, GitHub Pages 같은 정적 웹 호스팅에 올려야 합니다.
5. 현재 버전은 정적 HTML 프로토타입입니다. 회원가입, 로그인, 검사 데이터 누적 기능은 실제 서버와 데이터베이스 연결 후 작동합니다.
